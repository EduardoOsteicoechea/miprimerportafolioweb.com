La estructura básica de un documento de html

Toda página web (correctamente desarrollada) tendrá este código básico. 

	<!DOCTYPE html>
	<html lang="en">
		<head>
			<meta charset="UTF-8">
			<meta name="viewport" content="width=device-width, initial-scale=1.0">
			<title>Aquí va el título de la página web</title>
		</head>
		<body>
		</body>
	</html>

Puedes copiar y pegar este código en cada página o escribir "html:5" y presionar la tecla "enter" para generarlo automáticamente en vscode.

Parece intimidante, pero solo tiene 4 elementos.

  1. !DOCTYPE html: Le indica al navegador qué tipo de archivo es éste.
  2. html: Contiene el código de la página web que se ubica en 2 contenedores: "head" y "body".
  3. head: Contiene el código que configura algunos aspectos de la página como el título que aparece en las pestañas de los navegadores, el ícono al lado del título, la configuración de la página para adaptarse a diferentes dispositivos, los archivos que estarán vinculados a ésta página en particular e información útil para los motores de búsqueda.
  4. body: Contiene el contenido de la página web (en HTML).
