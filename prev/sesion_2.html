CSS en X ideas

  1. Definición

    CSS es una abreviación para Cascading Style Sheet.

    Esto significa Hoja de Estilos en Cascada.

    Css es una hoja o archivo que contiene los estilos de la página y los aplica comenzando en parte superior y bajandando a partir de allí.

  2. Estilos intralineales

    Una de las tres formas de aplicar CSS (Estilos) a una página.
    Se colocan en el atributo style de cualquier elemento de hmtl.

  3. Estilos en head

    Otra forma de colocar estilos es colocar una etiqueta style en el head y escribir css dentro de ella.

  4. Estilos en archivo .css

    La tercera manera (la recomendada) es crear un archivo de css y vincularlo a la página.

  5. Selectores

    Los selectores especifican qué elemento estaremos estilizando

    Los selectores son elementos de

    html,
    id's,
    clases
    selectores compuestos
    pseudolementos.
